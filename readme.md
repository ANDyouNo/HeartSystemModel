# Описание файлов
tests - графические результаты тестов модели
pkl - модель, обученная на данных
HeartGenSimple - базовый генератор датасета
testGen - генератор датасета для тестов с шумом и более реальными паттернами
simpleModelAllFeatures - скрипт для обучения модели по датасету
OnePeopleAnalyze - запуск анализа одного человека из датасета с использованием модели
